# SIS Website

## Installation

- In `/app`, create virtual environment with `virtualenv venv`, then install `flask` and `python-dotenv`.
- In `/` directory, make sure that `yarn` and `React`/`node`/`npm` are installed properly.

## To run use:

- `yarn start`
- `yarn start-api --port 8000`
